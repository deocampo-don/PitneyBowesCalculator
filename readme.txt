##### PURPOSE #####
This tool was created to make the work easier by reducing manual counting and paperwork. 
It also stores all envelope, tray, and pallet counts in a database, making it easier to track and generate reports when needed.

##### DESCRIPTION #####
This tool helps simplify envelope counting and reduce manual paperwork. Instead of counting envelopes by hand, users can build pallets and scan the Job or Work Order to automatically capture envelope counts.
Once all pallets are packed, the shipment is ready to go, and a PB Summary is printed showing all PB Jobs along with pallet, tray, and envelope counts.
